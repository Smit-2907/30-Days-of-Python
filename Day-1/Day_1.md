# 🐍 Day 1 — Python Basics (Foundation Day)

## 🎯 Goal
Understand how Python programs run and learn the basics:
- `print()` function
- variables
- data types
- simple program structure

---

# 🗺️ Day 1 Flow (Follow in Order)

### 1️⃣ Run Your First Python Program
Create a file: hello_world.py

Write:

python
print("Hello World")

Run the file and verify the output.

---

### 2️⃣ Practice the `print()` Function

Create: printing_demo.py

Write a program that prints:

Hello
My name is John Doe
I am learning Python

---

### 3️⃣ Learn Variables

Create: variables_demo.py
Create variables for:

- name
- age

Print them using `print()`.

---

### 4️⃣ Understand Variable Naming Rules

Practice creating valid variables like:

student_name
user_age
total_marks

Try writing invalid variables to see Python errors.

---

### 5️⃣ Learn Basic Data Types

Create: data_types.py
Create variables for:

- a string
- an integer
- a float

Print them to the console.

---

### 6️⃣ Print Variables Inside Sentences

Create: print_sentence.py

Write a program that prints sentences using variables like:

My name is John Doe
I am 25 years old

---

### 7️⃣ Solve the Exercises

Practice by writing small programs:

- Program to print three sentences
- Program that stores **name, age, city**
- Program that prints **learning Python for 30 days**


### 8️⃣ Mini Challenge

Create: student_intro.py

The program should print:
Student Name: John Doe
Age: 25
Course: Python
City: New York

### ⭐ Optional Bonus
Create: check_types.py
Use the `type()` function to check data types.
